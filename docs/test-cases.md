## 测试用例1：正常提交预约

### 前置条件
1. 学生已登录
2. 已填写首访登记表
3. 已签署知情同意书

### 测试步骤
1. 访问首访登记表页面（/student/first-visit-form）
2. 填写表单：
   - 主要困扰：学习压力
   - 问题详细描述：最近考试压力较大，睡眠不好
   - 期望帮助：希望获得压力调节建议
   - 情绪评分：6
   - 睡眠评分：5
   - 压力评分：8
   - 自伤风险：无
   - 紧急求助：否
3. 点击提交按钮
4. 自动跳转到知情同意书页面（/student/consent）
5. 阅读知情同意书内容
6. 勾选"我已阅读并同意知情同意书"
7. 点击"签署同意"按钮
8. 自动跳转到初访预约页面（/student/appointment-create）
9. 选择日期：2026-06-10
10. 选择时间段：上午第一段（08:30-09:20）
11. 点击"提交预约"按钮

### 预期结果
1. 首访登记表提交成功，显示成功提示
2. 知情同意书签署成功，显示成功提示
3. 预约提交成功，显示成功提示
4. 自动跳转到我的预约页面（/student/appointments）
5. 我的预约页面显示新创建的预约记录，状态为"待审核"
